# Library Management System

A simple **Python console application** to manage a collection of books.  
You can add, remove, search, and view all books in your library.

## Features

- Add a book with **title, author, and publication year**  
- Remove a book by its **title**  
- Search for a book by **title**  
- View all books in the library  
- User-friendly console interface with clear prompts and messages  

## How to Run

1. Make sure you have **Python 3** installed.  
2. Clone or download the repository
